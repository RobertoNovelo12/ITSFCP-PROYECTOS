Documentación del Sistema Web Responsivo para Gestión de Proyectos – ITSFCP

Introducción

Este sistema permite administrar, monitorear y controlar los proyectos académicos y de investigación del Instituto Tecnológico Superior de Felipe Carrillo Puerto. Desde esta documentación aprenderás a usar todas las funciones disponibles.

Acceso al sistema

Para acceder, ingresa tus credenciales institucionales. Si olvidaste tu contraseña, contacta al departamento de sistemas.

  • Pantalla de inicio de sesión limpia y responsiva.
  • Control de errores por credenciales incorrectas.
  • Redirección automática al dashboard.

Dashboard

Una vez dentro, verás un panel general con datos importantes:

  • Total de proyectos registrados.
  • Proyectos por estatus.
  • Acceso rápido a las funciones principales.

Gestión de Proyectos

Desde la sección Proyectos podrás:

  1. Registrar un nuevo proyecto.
  2. Editar información existente.
  3. Actualizar avances.
  4. Asignar docentes o estudiantes.
  5. Consultar detalles completos de cada proyecto.

Los proyectos cuentan con estatus como En proceso, Terminado, Pendiente, etc.

Generación de reportes

El sistema permite generar reportes por periodo, tipo de proyecto o estatus.

  • Filtro por fechas y categorías.
  • Visualización inmediata.
  • Opción para exportar (ver siguiente sección).

Exportación a PDF

Todos los reportes pueden exportarse a PDF con un diseño limpio para imprimir o archivar.

  • Formato responsivo optimizado para impresión.
  • Compatible con navegadores modernos.

Gestión de Usuarios

Los administradores pueden crear, editar o eliminar usuarios.

  • Roles: Administrador, Docente, Alumno.
  • Control de permisos según nivel.
  • Edición rápida desde la tabla de usuarios.

Soporte

Si tienes dudas o encuentras un error, contacta a soporte institucional.

Correo sugerido: moodle@itscarrillopuerto.edu.mx
